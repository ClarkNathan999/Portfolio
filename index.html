Particle Simulation

I was tasked with creating a particle simulation using OpenGL and GLUT. The intention was to learn more about animation methods, specificly Numerical Intigration. This porject simulated particles being affected by gravity, and a black hole using the Eulers Method. Initialy, the user is able to draw points on the window using mouse clicks, If the user hits the 'g' key a gravity simulation starts, and if the user hits the 'b' key a black hole simulation begins. If i had more time I would have liked to implement the black hole method using fourth order Runga kutta to make the simulation smoother and more accurate.
